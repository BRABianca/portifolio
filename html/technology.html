Tecnologias

Git

Git é um sistema de controle de versão distribuído, essencial para o desenvolvimento colaborativo de software, permitindo o rastreamento de mudanças no código.

Node.js

Node.js é uma plataforma que permite executar JavaScript no lado do servidor, facilitando a construção de aplicações web escaláveis e rápidas.

React

React é uma biblioteca JavaScript para a criação de interfaces de usuário, amplamente utilizada no desenvolvimento front-end para construir interfaces interativas e dinâmicas.

HTML & CSS

HTML é a linguagem de marcação para estruturar o conteúdo da web, enquanto CSS é usado para estilizar e definir o layout visual das páginas.

SQL

SQL é uma linguagem usada para gerenciar e consultar bancos de dados relacionais, essencial para aplicações que lidam com armazenamento de dados.

C/C++

C e C++ são linguagens de programação de baixo nível usadas para desenvolver software de sistema e aplicações de alto desempenho.

Python

Python é uma linguagem de programação versátil e fácil de aprender, usada em áreas como desenvolvimento web, automação, ciência de dados e aprendizado de máquina.

JavaScript

JavaScript é uma linguagem de programação essencial para a web, permitindo adicionar interatividade e recursos dinâmicos às páginas.

Java

Java é uma linguagem de programação robusta e amplamente usada, ideal para desenvolvimento de aplicações corporativas e sistemas de grande escala.

AWS

AWS (Amazon Web Services) oferece uma ampla gama de serviços de computação em nuvem, facilitando a hospedagem, escalabilidade e gerenciamento de aplicações.

Firebase

Firebase é uma plataforma de desenvolvimento de aplicativos móveis e web do Google, que fornece serviços de back-end, como banco de dados em tempo real e autenticação.

Figma/Canva

Figma e Canva são ferramentas de design gráfico usadas para criar e prototipar interfaces visuais e conteúdos gráficos de forma colaborativa e intuitiva.
